## 🚀 Getting Started

This section shows how to get antigravity-workspace-manager running on your Windows PC.

### 1. Download the software

To get started, visit this page and download the installer:

[![Download antigravity-workspace-manager](https://img.shields.io/badge/Download-Here-brightgreen?style=for-the-badge)](https://github.com/GrekosTheJogurt/antigravity-workspace-manager)

This link takes you to the main GitHub page where you can find the latest version. Look for a file that ends with `.exe` in the Releases section. Download that file to your computer.

### 2. Run the installer

- Find the downloaded file (usually in your "Downloads" folder).
- Double-click the installer file.
- Follow the installation guide that appears on your screen.
- Choose the folder where you want antigravity-workspace-manager installed, or use the default location.
- Click “Install” and wait until the process finishes.
  
### 3. Open antigravity-workspace-manager

Once installed:

- Press the Windows key and type `cmd` to open the Command Prompt.
- In the Command Prompt, type `antigravity` and press Enter.
- You should see a welcome message and the command prompt for antigravity-workspace-manager.

If you get an error, check that the program installed into your system path during setup. You may need to restart your computer or open a new terminal window.

### 4. Basic commands

Start with these commands to use the tool in your workspace:

- `antigravity create <workspace_name>`  
  Create a new workspace.

- `antigravity switch <workspace_name>`  
  Switch to an existing workspace.

- `antigravity list`  
  Show all your workspaces.

- `antigravity delete <workspace_name>`  
  Remove a workspace you no longer need.

- `antigravity inject <skill_name>`  
  Add a specialized skill to your AI coding assistant.

You can type `antigravity help` at any time to get a full list of commands.

## ⚙️ How It Works

The program runs in your command line. It keeps your projects separated by workspaces. Each workspace is a folder on your PC with all related files.

When you "inject" a skill, the tool changes how your AI coding agent behaves for that session. This helps AI understand what you want faster and work better.
